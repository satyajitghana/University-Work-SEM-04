Marks Predictor

Predicts marks of the student from the data given by the previous tests scores, namely, Term Test 1, 2, 3 and Assignment 1, 2.
1. Input your marks in proper numeric format, leave a 0 if you were absent for the exam
2. Add additional subjects if required by clicking on Add subject
3. Name the Subjects with their Subject name or the Course Code
4. Click on Predict and wait for the model to train and predict your marks

Attendance Predictor

Predicts the attendance of the student if the student has a possibility to be eligible from the current attendance percentage.
1. Input your current attended number of classes
2. Input the current total classes, which is number of classes you missed + current attended
3. Input your target percentage attendance 4. Input the total number of classes or at least approximate 5. Click on Predict
